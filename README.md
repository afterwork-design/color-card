# [color-card](https://afterwork-design.github.io/color-card/)

卡卡色板，在线地址 [https://afterwork-design.github.io/color-card/](https://afterwork-design.github.io/color-card/)。

## 本地运行

首先需要安装 node.js，建议 14.18.0 版本。

1、克隆仓库
2、安装依赖：`npm install`
3、运行：`npm run dev`
